# ASUS TUF Gaming FX506L Teardown and Upgrade Assessment

## Overview

This project documents the complete teardown of an ASUS TUF Gaming FX506L gaming laptop. The objective was to identify internal components, understand the hardware layout, evaluate serviceability, and assess potential upgrade opportunities.

---

## Device Information

* **Model:** ASUS TUF Gaming FX506L
* **Manufacturer:** ASUS
* **Device Type:** Gaming Laptop
* **Graphics:** NVIDIA GeForce GTX 1650
* **Memory (RAM):** 8 GB DDR4
* **Storage:** 512 GB M.2 NVMe SSD
* **Battery:** 48 Wh Lithium-Ion Battery
* **Connectivity:**

  * Wi-Fi
  * Bluetooth
  * Ethernet (RJ-45)
  * HDMI
  * USB Ports
* **Cooling System:** Dual-Fan Cooling with Heat Pipes

---

## Disclaimer

This project focuses on laptop disassembly and component identification. No hardware upgrades were performed during this project. Upgrade recommendations are based on the hardware discovered during teardown.

# Conclusion

The ASUS TUF Gaming FX506L was successfully disassembled and documented. Major internal components were identified, including the motherboard, cooling system, storage device, wireless networking hardware, battery, and display assembly. The teardown revealed several upgrade opportunities, particularly for RAM and NVMe storage, while also demonstrating that key serviceable components such as the battery, cooling fan, keyboard assembly, and display assembly can be replaced when necessary. This project provided valuable insight into laptop hardware architecture, repairability, and upgrade potential.
